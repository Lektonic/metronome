# Terminal Metronome

```bash
$ ./metronome --help

Usage: metronome [OPTIONS]

Options:
  -t, --tempo <BPM>          [default: 60]
  -b, --bpb <BEATS_PER_BAR>  [default: 4]
  -h, --help                 Print help
  -V, --version              Print version
```

A very simple visual metronome tool.

Prints a dot for every beat.

Might one day make noise as well.
